Share Assets

To create a element that would share assets of particular asset type from one side to other.
Output : errors(if any), total # of shared assets, assets (id : name) format

GENERAL STEPS TO FOLLOW:

  1. Enable the asset type at target website
  2. Count # of assets before sharing from ASSETPUBLICATION table
  3. Share asset (using asset:share tag)
  4. Count # of assets after sharing from ASSETPUBLICATION table
  5. pass c, cid and run=true from site entry (if run=false, or missing from site entry, assets won't be shared).
